# MedRemind

Medication reminder app (React Native + Expo).

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm start
```

3. Run on a device or simulator:

- Android: `npm run android`
- iOS: `npm run ios`

## Notes

- Assets (app icon, splash) are expected under `assets/`.
- This project uses Expo SDK 49.
